闰年的判断

判定某年为闰年的标准是：

  • 能被4整除且不能被100整除的
  • 能被400整除的

用C语言实现代码就是

1
2
3
4
5
if ((year % 100 != 0) && (year % 4 == 0) || (year % 400 == 0)) {
   //输出该年是闰年;
} else {
   //输出该年是平年;
}

一种更简单的写法是

1
#define ISLEAP(yr) ((!((yr) % 4) && (yr) % 100) || !((yr) % 400))
